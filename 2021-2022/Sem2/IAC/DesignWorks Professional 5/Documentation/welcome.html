Access Codes

You will need a separate access code to run the Simulator option. This means you will have to have your base version 5 access code (15 letters and numbers) that you got with the product plus a second code for the simulator option. These two codes need to be entered one per line in the Register box in the program. You may need to restart the program after entering the codes to get it to load the new modules.

To Remove the Simulator Option

You can remove the simulator option by using the Remove icon that the installer places in the DesignWorks Professional submenu in the Start menu, then reinstall the schematic package from a regular installer.

To manually remove the simulator code files you can do the following:

  • delete all files from the Tools folder in the DesignWorks program folder
  • reinstall DesignWorks Professional from a non-simulator version installer

File Formats

The file format for design files (.CCT) has been updated to accommodate the new graphic drawing tools. Files created or modified in version 5 will not be readable by older versions.

The library file format (.CLF) has not changed.

Installing and Opening Symbol Libraries

DesignWorks comes with a variety of different symbol library files and many more can be downloaded from our web site. By default, only a small number of these are opened automatically at startup. You can open and close libraries manually or customize which libraries open automatically when the program is started, to suit your application.

NOTE: DesignWorks symbol library files normally have the ".CLF" file name extension.

Location of Libraries

When DesignWorks is first installed, the libraries will be located in the Libs folder inside the DesignWorks program folder. Inside that folder may be other sub-folders containing additional families of libraries.

Opening Libraries Manually

To open library files manually, right click in the parts palette window, or move to the Libraries submenu of the File menu, and select the Open Lib command. This command allows you to locate a symbol library file in the usual way. When the file is opened, it will be added to the list of available libraries in the parts palette.

Opening Libraries Automatically at Startup

When you use the Open Lib command described above, you can check the box “Open automatically at startup” to add this library to a list to be opened each time the program starts. To manage this list, you can use the Auto Open Libs command.

If you wish to create a more sophisticated layout of folders containing libraries for your projects, you can use the LIBRARY and LIBRARYFOLDER keywords in the DesignWorks initialization file (DW.INI) to create additional library folders. For more information, look up the LibraryFolder keyword in the index of the DesignWorks manual.

Removing Custom Menus

The package comes pre-configured with a custom menu that demonstrates the scripting and customization features. To remove this or any existing custom menu, open the DW.INI file using Notepad, and look for any lines starting with "menu =" (keywords are not case sensitive, so it could be "Menu" or "MENU"). Comment these lines out by inserting "//" in front of them.
