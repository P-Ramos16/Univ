The DesignWorks Object Model

When a JavaScript is run within DesignWorks, a number of objects are available representing the various elements of a design and top level data and functions in the program. Each of these objects has a set of properties and methods that can be used to access and modify data and invoke program functions under script control.

This table summarizes the available object types that are directly related to design data:

DWAttribute

Represents an attribute value associated with a circuit, device, signal, pin, type, or type pin.

DWCircuit

Represents a circuit structure, which may be the top level circuit in a design (or the whole design, if no hierarchy is used) or a subcircuit of some device in a design.

DWDevice

Represents a device instance on a diagram. This includes “pseudo devices” such as page connectors, bus breakouts, etc.

DWLibrary

Represents a symbol library file

DWPage

Represents a page in a circuit

DWPic

Represents a picture, i.e. a collection of graphic elements like lines and rectangles, usually used for device symbols.

DWPin

Represents a single pin on a device

DWSignal

Represents a signal in a circuit, i.e. a list of interconnected device pins and the associated visible line segments.

DWStatic

This is a top-level object that always exists by default and is used to access global program data and functions.

DWType

Represents a device type, i.e. a device symbol and the associated default attributes, etc.

DWTypePin

Represents a pin on a device type

DWWindow

An object used to draw interactively into specific areas of the screen, e.g. a device symbol, in order to provide interactive displays

The following additional objects are available that provide utility functions like file and database access, alerts and dialogs, etc.

SPArray

A general-purpose array object used to pass arrays of other objects and strings to and from other objects

SPDatabase

An object that allows read/write access to an Access database

SPFile

An object that represents a disk file

SPFTP

Represents an FTP connection to a remote system over the Internet

SPHTMLDialog

Represents a multi-page dialog displayed using HTML

SPInternet

Represents an HTTP connection to the Internet

SPStringMap

A general-purpose associative string array object
