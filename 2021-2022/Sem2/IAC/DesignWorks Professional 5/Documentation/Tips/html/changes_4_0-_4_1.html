DesignWorks Professional for Windows Version 4.1

The primary focus in this release is to add specific schematic editing features requested by users since the 4.0 release..

New Schematic Editing Features

  • Drag and drop devices from parts list to schematic
  • Create Subcct Block command. This allows you to quickly convert a circuit or group of selected items into a subcircuit block with just a few mouse clicks.
  • New Port Connector command.  This makes quick work of adding port connections to a subcircuit block, especially for bus pins.
  • Edit Symbol command. Just right click on a symbol in a schematic to launch straight into the symbol editor to make changes.
  • New order options to Repackage Design. Repackage Design can now be set to apply names in any selected order.
  • Improvements to Update from Lib command. New options and less prompting with unneeded warnings and options.
  • Read Only option in Define Attribute Fields
  • Added scriptable attribute probe.  You can now customize the information that appears when the attribute probe is clicked on an item.
  • More colours in INI file

    • Back Annotation

  • Added part number field selector for PADS format back annotation

    • Export

  • All lines now limited to the size specified in $LINEWIDTH
  • Changed default $LINEWIDTH to 32000

    • General Changes

  • Numerous small bug fixes and fine tuning
