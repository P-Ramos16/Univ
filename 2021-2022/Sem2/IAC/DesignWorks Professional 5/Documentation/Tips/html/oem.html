Schematic Capture for OEM Applications

If you produce circuit or network analysis, layout or simulation software and need a graphical diagramming tool as a front end, consider DesignWorks Professional.  DesignWorks has numerous customization features that allow the package to be configured for your application, often without writing a single line of program code!  Custom menus, symbol libraries, netlist and report outputs, design templates and other items can be customized to meet your requirements.

  • For an example of a major new product based on DesignWorks Professional, see the EMTP power systems analysis tool.
  • Contact us to discuss your application.

>For some applications, the customization tools provided with the basic package provide all the flexibility you need to build the perfect tool for your situation. In other cases, new features, new packaging or rebranding may be needed, all of which are possible. The following sections provide more details on these options.

Customizing the Standard DesignWorks Package

The standard package includes the ability to create custom symbol libraries, design templates, export reports, error checking scripts, custom menus and some other items. For many applications, modifying these is all that is needed, without making any changes to the program itself. You can just resell our regular package and send the customer a "design kit" containing the extra items they need. We would work out a reseller discount based on volume and on the amount of support required.

Rebranding and Custom Installers

The next level would be to make small changes to the program, such as a custom installer containing only your items, and possibly rebranding the package with your name and logo. Also, we have some enhanced scripting capability that we do not normally ship with the basic package that allows some very sophisticated data entry, property sheets, error checking and other features using JavaScript and HTML.

Full Customization

Lastly, would be some significant customization of the software, which would be required to add drawing or graphical features specific to your application. As an example of this, see the power systems simulation package EMTP. The front end, sold as EMTPWorks, is actually DesignWorks Professional with quite a few custom modifications.
