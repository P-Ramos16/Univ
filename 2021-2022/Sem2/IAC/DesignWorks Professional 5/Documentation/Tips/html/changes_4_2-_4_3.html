DesignWorks Professional for Windows Version 4.3

The primary focus in this release is to enhance and upgrade the DesignWorks Digital Simulator Option, but there have also been some small enhancements and fine tuning to other parts of the program, listed below

Digital Simulator Enhancements

  • Support for high-level-language modeling using a subset of VHDL
  • All delay time values are specified in real time units with femtosecond precision
  • All signal values can be displayed simultaneously on the schematic
  • New Model Wizard command simplifies creating and attaching simulation models using structural schematics or VHDL.
  • Many enhancements to the timing diagram display to support VHDL enumerated types, etc.

Schematic Editing Enhancements

  • Extended pin numbers to 8 chars and names to 32
  • Print & Print Setup in circuit popup menu
  • Popup menus for text and picture objects
  • Double-click on text to open for editing
  • Added custom command items in popup menus
  • Apply grid to auto-positioned signal names
  • Save to Lib added "use rotated image" option
  • Added Scope option to Make Unique
  • Edit Symbol now displays option to edit rotated symbol
  • Show temporary fields in Define Attribute Fields
  • Allow "temporary" setting to be toggled in Define Attribute Fields
  • Added separate text font settings for attribute fields
  • New font background and frame options
  • Added new message box for read only circuits
  • Change signal auto-naming to use name table
  • Allow drag and drop from preview window
  • Fine tune placement of signal names

INI File & General Operation

  • Added: -hide command line option
  • Added: Variable substitution in directory names in INI
  • Added: File path variables %my documents% %program files% %windows% %desktop%
  • Show nested example folders in Examples command

Symbol Editor & Libraries

  • Take library read-only status into account in library dialogs
  • Library part name filter accepts *, ?, # pattern chars
  • Added Select All command in pin list popup
  • Add check for bus internal pin order before saving
  • Extract pins automatically when subcct attached

Export

  • Added $ISDIRTY $ISSAVED
  • Added: accept arguments for -exp command line option
  • Added: accept arguments in $RUNSCRIPT
  • Added: $GETENV command

 

Better yet, version 4.3 is a free upgrade for all registered version 4 users.

How to get the new version:

  • Registered DesignWorks Professional 4 users: please go to the Registered User’s Page to download your full upgrade.
  • New users, go to the Downloads page to download the latest evaluation version.
