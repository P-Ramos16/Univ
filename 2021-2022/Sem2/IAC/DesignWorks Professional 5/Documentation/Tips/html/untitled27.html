Copy and Paste is a great way to move a design or a portion of one into Word. Just select the desired portion of the circuit by

  • using the pointer to draw a selection rectangle around the design items
  • using the Select All command in the Edit menu
  • clicking on the design items using the SHIFT key

Use the Copy command in the Edit menu to place the items on the clipboard, then switch to Word and select Paste Special and select the Picture option.

Note: If any text items were selected, DesignWorks places text on the clipboard and Word will place that by default if you use its Paste command, so the Paste Special command is more reliable.
