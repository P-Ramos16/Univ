[DesignWorks 5 New Features]

When DesignWorks is first installed, it is configured to open automatically any library file that it finds in the Libs folder inside the DesignWorks program folder or in the alternate location My Documents\DesignWorks\Libs. You can add more libraries to be opened automatically by simply placing them (or a shortcut to them) in this folder. Sub-folders are NOT scanned, so files that are not immediately needed can be "hidden" in nested folders.

You can also select any library in any location to be opened at startup by opening it with the Open Lib command and checking the “open automatically” checkbox.

If you wish to create a more sophisticated layout of folders containing libraries for your projects, you can use the LIBRARY and LIBRARYFOLDER keywords in the DesignWorks initialization file (DW.INI) to create additional library folders. For more information, look up the LibraryFolder keyword in the index of the DesignWorks manual.
