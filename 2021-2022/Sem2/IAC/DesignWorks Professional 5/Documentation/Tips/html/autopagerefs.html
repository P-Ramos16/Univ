Automatic Page References

DesignWorks Professional 4 has an Automatic Page Reference facility which displays the page number and position of any Page Connectors associated with a given signal.  These page references are automatically displayed immediately adjacent to any Page Connector as soon as multiple Page Connectors are attached to the same signal.

Here are some of the features of the auto page reference facility:

  • As long as automatic updating is enabled, these references will be updated automatically when any Page Connectors or attached signals are added, deleted, moved or renamed.
  • Automatic page reference updating can be enabled/disabled for the design using the Design Preferences command.
  • Page references can be updated manually using the Update Now button in the Design Preferences box.
  • The page reference format can be set using the Design Preferences command.
