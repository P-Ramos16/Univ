Tip Ideas

38) In the "Device Editor" only attribute fields that have a value are stored with the symbol. When you save the symbol
to a library, any fields with null values are stripped off to save storage. 
 
39) You can use any folder containing circuit files as a “library”
 
40) Creating a device symbol can involve much more than just drawing the graphics that represent the device. Many DesignWorks features and many features in other ECAD packages that you may want to interface with, rely on correct text attributes being associated with each symbol. Please see chapter six in the manual for more information.
 
41) You can elect globally to show the field name for a certain attribute whenever it is used. See "Defining a New Attribute Field" in the manual.
 
42) The "Name" attribute is a special case in that it can be displayed by simply clicking the text cursor on the object in question.
 
43) You can create custom menu items that run scripts or open files
 
44) When placing a device symbol, you can determine its orientation in advance using the "Orientation" command or the arrow keys on the keyboard.

24)You can adjust the position of all the objects on a page by using 
the "Centre in Page" command. This command moves all items on the 
current page so that the circuit objects are centered in the page.
