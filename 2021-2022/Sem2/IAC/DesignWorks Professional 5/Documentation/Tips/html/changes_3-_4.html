New Features in Version 4

Here is a summary of the new features in version 4, for users of version 3.x:

  • All new user interface - dockable windows, workbook view, improved toolbars, more menu options
  • Symbol preview
  • DXF output
  • Scriptable Find
  • Clear Change Count
  • Attribute "Never Visible" option
  • Report generation a la Mac v4
  • Sheet Size Wizard and Border Wizard simplify sheet setup
  • "standard" sheet sizes definable in INI file
  • Fit to Printed Page option
  • Hierarchy Wizard
  • Nudge
  • ClipboardColors ini keyword
  • Timed autosave
  • Part Info command in library popup
  • Built in text document viewer
  • New Lib and Open Lib buttons on part save dialog
  • Custom menus
