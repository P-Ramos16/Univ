DesignWorks Professional 4 for Windows
version 4.2 Now Available!

Most of the changes in this release are in the symbol editor, with a few other small improvements and fine tuning.

Device Symbol Editor:

  • Optionally display pin name with pin
  • Symbol Gallery: A new window gives you quick access to frequently used symbols, special pins and graphics
  • New Magnify command and improvements to zoom in/out
  • Popup context menus on all objects
  • Properties command command to all objects
  • Rotate/flip all objects
  • Group/Ungroup can include pins
  • PinName and BusName INI keywords
  • Open Subcircuit command allows direct access to subcircuit stored with symbol
  • Browse button in Subcct & Part Type
  • New commands: Show/Hide Parts List, Pin List, Symbol Gallery
  • | notation to specify pin function in Auto Create Symbol

Schematic Editing:

  • Added ANALOG pin type
  • Update from Lib, Attributes Only option
  • Added Bus Pin Options to pin info box
  • Added Add/Delete Pins to Bus Pin Options
  • Added signal list to bus info
  • UndoLevels INI keyword
  • Paste Special command
  • Contents of all selected misc text items are copied onto clipboard as text

General Operation:

  • Allow environment variable and registry references in file names
  • Improvements to text editor
  • Specify alternate text file types in INI file
  • Help/Root Folder command allows an alternate folder to be specified as the program starting folder. This is intended to allow the program to be run from read-only or protected servers.

 

Better yet, version 4.2 is a free upgrade for all registered version 4 users.

 

How to get the new version:

New users, click here to download a free evaluation version of the current release.. Registered version 4 users (anyone with a valid DesignWorks Professional 4 access code): You can either:

  • Download and install the regular evaluation version.  This contains all program elements that have been updated.  You can simply install the new version on top of your existing package (that is, into the same folder), and your existing access code will unlock it into a permanent package.
  • If you want to redownload the full program installer, have your access code (15 letters and numbers) handy and go to the registered users page.

Registered version 3 users (have a version 3 access code consisting of unit number and password(s)): Go to the upgrade status page and find out if you qualify for a free upgrade to version 4 or arrange to purchase an upgrade.
